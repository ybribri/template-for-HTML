HOW TO USE
DOWNLOAD
Download a file from here, and put it in HTML head.
<
script 
src
=
"
etemplate.js
"
>
<
/script
>
MAKE HTML OR CSS
Make your own HTML file or CSS file(or CSS in style tag) using templates.
<span>
<%%= et.names[0] %>
</span>

<
div 
>
<%% include('/module/_header.html') %>
<
/div
>

const  etm = new  eTemplate () ;

};

etm . render ();
< /script >
