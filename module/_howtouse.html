HOW TO USE
DOWNLOAD
Download a file from here, and put it in HTML head.
<
script 
src
=
"
etemplate.js
"
>
<
/script
>
MAKE HTML
Make your own HTML file using templates.
<span>
<%%= et.names[0] %>
</span>

<
div 
>
<%% include('/module/header.html') %>
<
/div
>
INSERT SCRIPT WITH DATA
Place eTemplate() with data object inside script tag
< script >
eTemplate ({
data:  {
subject: [ "mathematics" , "biology" , "physics" ],
data:  [ 3 , 4 , 5 ]
}
});
< /script >
