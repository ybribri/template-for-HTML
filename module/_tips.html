To render other page on current page, use <a> tag event-handler in eTemplate() with sync_url.

Be careful:  When a new file is rendered,
• old css files would be replaced to new css files
• new html would be rendered.
• etemplate.js and function addListener() are still in effect.
   You don't have to declare them in a new file again.

Be careful:  use #filename(# + filename) for href in <a> tag, like <a href="#feature">
• eTemplate will take "url/#feature" as same with "url/feature.html".
• Browser's previous or next button doesn't trigger click event even though hash is changed.
